Academic Curriculums

The Four Pillars of InsideFactz Training

Our learning progression leads you sequentially from analytical math spreadsheets to developing custom artificial intelligence agents.

Track 1

MS Excel Foundations

Master spreadsheets for business reporting, mathematical operations, and interactive charting dashboard building.

Syllabus Highlights

  • Spreadsheet mastery and cell referencing
  • Advanced formulas (VLOOKUP, SUMIFS, INDEX)
  • Pivot Tables, KPI Cards, and Dynamic Slicers
Weeks 1 - 3
Track 2

Python & Data Science

Move beyond spreadsheets. Write code, clean datasets, manage relational databases with SQL, and run models.

Syllabus Highlights

  • Core Python programming (Loops, OOP, Lambda)
  • Pandas & NumPy data wrangling pipelines
  • SQL database modeling & Seaborn visual analytics
Weeks 4 - 8
Track 3

Machine Learning

Learn predictive model building, classical machine learning algorithms, text mining, and deep learning basics.

Syllabus Highlights

  • Supervised algorithms (XGBoost, Decision Trees)
  • Unsupervised clustering & Text NLP mining
  • Deep Learning Neural Networks & TensorFlow models
Weeks 9 - 13
Track 4

Generative AI

Unlock advanced AI technologies. Build custom cognitive agents, vector databases, and RAG pipelines.

Syllabus Highlights

  • Prompt Engineering & LLM APIs (GPT, Claude)
  • Retrieval Augmented Generation (RAG) endpoints
  • LangChain orchestration and Vector DB integrations
Weeks 14 - 16
